## 开发规范

- **技术栈**：微信小程序原生开发，JavaScript + WXML + WXSS
- **UI组件**：使用 Vant Weapp 组件库（项目已引入）
- **样式风格**：延续项目现有清新活泼风格，保持一致的配色和布局
- **代码风格**：ES6+ 语法，`const`/`let`，严格相等，驼峰命名，`async/await` + `try/catch`
- **错误处理**：网络请求失败需要友好提示用户
- **加载状态**：页面加载时显示loading

## 接口调用方式

所有云函数调用方式统一为：
```javascript
wx.cloud.callFunction({
  name: '云函数名',
  data: { ...参数 }
}).then(res => {
  const { result } = res
  if (result.success) {
    // 处理成功
  } else {
    wx.showToast({ title: result.error || '操作失败', icon: 'none' })
  }
})
```
