项目信息：

- 现有功能：用户输入两个人姓名和关系类型，调用字节跳动豆包AI API生成关系匹配分析报告，支持广告变现、微信支付、分享裂变等商业模式

## 配置项

请在 `utils/config.js` 中增加：
```javascript
// 动态功能配置
module.exports.FEED_REWARD_TIMES = 1      // 每次发布奖励次数
module.exports.FEED_REWARD_MAX_DAILY = 2  // 每日最多奖励次数
```

## 奖励机制

发布动态成功后，**无论是匹配结果动态还是图文动态**，都奖励用户 `FEED_REWARD_TIMES` 次免费次数，每日最多奖励 `FEED_REWARD_MAX_DAILY` 次。请在 `createFeed` 云函数中处理奖励逻辑（统计今日用户已发布数量，不超过上限则奖励）。

## 图片处理

图文动态发布流程：
1. 用户在前端选择图片 → 调用 `wx.cloud.uploadFile` 上传到云存储 → 获取 fileID
2. 所有图片上传完成后，将 fileID 数组随其他参数一起调用 `createFeed` 云函数
3. 云函数创建动态记录入库
